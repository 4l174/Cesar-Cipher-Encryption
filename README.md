##Cifra de César

###Descrição
Este repositório contém uma implementação simples da Cifra de César em Python. A Cifra de César é uma técnica de criptografia clássica que envolve a substituição de cada letra em um texto por outra letra deslocada por um número fixo de posições no alfabeto.

###Funcionalidades
Cifragem: Implementação para cifrar mensagens usando a Cifra de César.
Facilidade de Uso: Interface simples para entrada de texto e exibição de resultados.

###Como Usar
1- Clone o repositório para o seu ambiente local.
2- Execute o script Python criptografia_de_cesar.py
3- Siga as instruções para cifrar uma mensagem.
